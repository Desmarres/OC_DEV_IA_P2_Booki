# Booki

Projet réalisé dans le cadre de la formation **Développeur IA**.

## À propos du projet

Booki est une plateforme permettant aux utilisateurs de trouver des hébergements et des activités dans la ville de leur choix.

L'objectif de ce projet était d'intégrer l'interface utilisateur d'après une maquette fournie par l'équipe de conception en utilisant uniquement **HTML** et **CSS**, tout en respectant les spécifications techniques et graphiques.

## Objectifs de la mission

* Intégrer une interface web à partir d'une maquette.
* Structurer une page web avec HTML5.
* Mettre en forme le contenu avec CSS3.
* Créer une mise en page responsive.
* Utiliser Flexbox pour l'organisation des éléments.
* Respecter les bonnes pratiques d'intégration web.
* Maintenir une structure de projet propre et organisée.

## Technologies utilisées

* HTML5
* CSS3
* Flexbox
* Git
* GitHub

## Fonctionnalités

* Barre de recherche de destination.
* Affichage des hébergements disponibles.
* Présentation des activités proposées.
* Section des hébergements les plus populaires.
* Interface responsive adaptée aux ordinateurs, tablettes et mobiles.

## Maquette

L'intégration a été réalisée à partir d'une maquette fournie dans le cadre du projet, avec pour objectif de reproduire fidèlement le design et l'expérience utilisateur attendus.
